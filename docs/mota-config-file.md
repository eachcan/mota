# MOTA 配置文件

MOTA 配置文件使用 JSON 格式，示例：

```json
{
    "includePath": [
        "C:\\path\\to\\include",
        "C:\\path\\to\\include2"
    ],
    "items": [
        {
            "output": "C:\\path\\to\\output",
            "source": "C:\\path\\to\\source",
            "lang": "cpp"
        }
    ]
}
```
