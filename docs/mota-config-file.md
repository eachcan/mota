# MOTA 配置文件

MOTA 配置文件使用 JSON 格式，示例：

```json
{
    "include_paths": [
        "C:\\path\\to\\include",
        "C:\\path\\to\\include2"
    ],
    "projects": [
        {
            "output_dir": "examples/generated",
            "source_dir": "examples",
            "lang": "yima-cpp"
        }
    ]
}
```
